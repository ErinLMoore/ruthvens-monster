# ruthvens-monster

### An extremely under construction game written in PhaserJS

-------------------

Controls:

+ right/left arrow: walk
+ down arrow: crouch
+ up arrow: rear up
+ up near climbable block: climb
+ space: bite attack
+ space near dead enemy: eat (gain back hp)
+ down arrow + space: claw attack
